# Nestlé-Style FMCG Sales and Demand Analysis

This project analyzes FMCG-style retail transactions using Python, SQL, and Tableau to understand category performance, sales channel contribution, city-level demand, and monthly sales trends.

## Tools Used
- Python
- SQL (MySQL)
- Tableau Public
- Excel / CSV

## Project Overview
The raw supermarket sales dataset was transformed into a Nestlé-style FMCG portfolio by creating business-relevant fields such as:
- Nestlé-style product categories
- Sales channels
- Demand levels
- Month and day features

## Key KPIs
- Total Sales: 322,966.75
- Total Gross Income: 15,379.37
- Total Quantity Sold: 5,510
- Top Category by Sales: Packaged Foods & Beverages
- Top Sales Channel: Modern Trade
- Top City by Sales: Naypyitaw

## Key Insights
- Packaged Foods & Beverages was the top-performing category by sales.
- Modern Trade contributed more sales than General Trade.
- Naypyitaw had the highest city-level sales.
- Medium-demand transactions were the most common.
- Sales peaked in the early months, indicating possible seasonal demand concentration.
